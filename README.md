# trabalho-avaliativo-bd

Este repositório contem um trabalho avaliativo com parte da funcionalidades do sistema da elaboração de casos clinicos, contendo as tabelas instituição,curso e professor

📌 Sobre o Projeto
O tema escolhido para este trabalho foi parte de um Sistema de Gestão de Geração de Casos Clinicos.

🛠️ Tecnologias Utilizadas
Linguagens: Java 17
Banco de Dados: PostgreSQL
Drivers de Conexão: JDBC (Java) / jdbc:postgresql
Interface: Console (Terminal)
