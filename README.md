# trabalho-avaliativo-bd

📌 Sobre o Projeto

O tema escolhido para este trabalho foi parte de um Sistema de Gestão de Geração de Casos Clinicos.

🛠️ Tecnologias Utilizadas

Linguagens: Java 17

Banco de Dados: PostgreSQL

Drivers de Conexão: JDBC (Java) / jdbc:postgresql

Interface: Console (Terminal)

Executando em Java

Importe o projeto na sua IDE (VS Code, IntelliJ,Eclipse ou Netbeans).

Certifique-se de ter o driver JDBC no classpath.

Execute a classe Main.java.
